- не менше трьох закритих змінних-членів;
- відкриті геттери і сеттери для кожної змінної;
- конструктор за замовчуванням і конструктор з параметрами;
- деструктор, який виводить повідомлення про знищення об’єкта;

- створіть кілька об’єктів класу;

## Відповіді на контрольні запитання

1. **Що таке клас і об’єкт?**

    Клас — це шаблон, який описує властивості (поля) та поведінку (методи) об’єктів.

    ```cpp
    class Car {
        int speed;
    public:
        void drive();
    };
    ```

    Об’єкт — це конкретний екземпляр класу.

    ```cpp
    Car myCar;
    ```

2. **Для чого потрібні специфікатори доступу?**

    Специфікатори доступу визначають, хто може звертатися до членів класу.

    - **public** — доступний усім  
    - **private** — доступний тільки всередині класу  
    - **protected** — доступний класу та його нащадкам  

    Вони дозволяють приховувати реалізацію та захищати внутрішні дані.

3. **Як реалізується інкапсуляція в C++?**

    Інкапсуляція — це приховування даних і контроль доступу до них.

    Реалізується через:
    - приватні змінні (`private`)
    - публічні методи (`public`)
    - геттери/сеттери

    ```cpp
    class BankAccount {
    private:
        double balance;

    public:
        void deposit(double amount) { balance += amount; }
        double getBalance() const { return balance; }
    };
    ```

4. **Яка роль конструктора та деструктора?**

    Конструктор:
        Автоматично викликається при створенні об’єкта.  
        Призначення: ініціалізувати поля, виділити ресурси.

    ```cpp
        Car() : speed(0) {}
        ```

    Деструктор:
        Автоматично викликається при знищенні об’єкта.  
        Призначення: звільнити ресурси.

    ```cpp
        ~Car() {
            delete[] buffer;
        }
        ```

5. **Що таке вказівник `this` і як він використовується для ланцюжкових викликів?**

    `this` — це вказівник на поточний об’єкт.

    ```cpp
    this->x = 5;
    ```

    Ланцюжкові виклики (method chaining) - Методи повинні повертати *посилання на поточний об’єкт*.

    ```cpp
    class Point {
    private:
        int x, y;

    public:
        Point& setX(int value) {
            x = value;
            return *this;
        }

        Point& setY(int value) {
            y = value;
            return *this;
        }
    };
    ```
